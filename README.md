## :bulb: Introduction 

A turn-based RPG game from Star Wars with (Docker, ReactJS, AdonisJS) (Ngnix, Reactjs, Redux, Redux Thunk, React Router, AdonisJS, PostgreSQL, Docker, PM2) + (Conventional Changelog with Commitlint, Husky, Git CZ)

## :memo: Minimal Requirements

* NodeJs 12.x
* Docker 18.x
* Docker-compose 1.25.x

## 🚀 Install

```
$ https://github.com/gabriel-roque/starwars-battle
$ cd starwars-battle
```

## :wrench: Configure in development (first time)

```
$ ./entrypoint-dev.sh
```

## :wrench: Configure in production (first time)

```
$ ./entrypoint-prod.sh
```

## 📈 Run app

```
$ docker-compose up -d
```

> Check **web-app** in http://localhost:3000/  for _**DEVELOPMENT**_ 

> Check **web-app** in http://localhost/ for _**PRODUCTION**_

> Check **api** in http://localhost:8080
